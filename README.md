# AI-Resume-Screening-System

## 📌 Project Overview
**AI Resume Screening System** is a Machine Learning and NLP-based web application that automates resume classification and analysis. Built using Python, Streamlit, TF-IDF, and ML models, it predicts suitable job roles, analyzes resume strength, provides confidence scores, visualizations, and downloadable reports.

---

## 🚀 Features
- 📄 Upload resume in PDF format
- 📝 Paste resume text manually
- 🤖 AI-based resume classification
- 📊 Resume strength analysis
- 📈 Confidence score prediction
- 🎯 Job role matching system
- 📉 Probability distribution charts
- 📥 Downloadable analysis report
- 💡 Smart recommendations for resume improvement

---

---

## ⚙️ System Workflow
1. User uploads resume or pastes text
2. Resume text is extracted and cleaned
3. TF-IDF converts text into numerical vectors
4. Machine Learning model predicts job category
5. System displays confidence score and analysis
6. User can download the generated report

---

## 🧠 Machine Learning Models
The project compares multiple ML algorithms:

| Model | Accuracy |
|-------|-----------|
| Naive Bayes | 64% |
| Logistic Regression | 74% |
| Support Vector Machine (SVM) | 72% |

✅ Logistic Regression achieved the highest accuracy.
